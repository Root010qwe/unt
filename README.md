# Как запустить проект на GitHub Pages

Этот проект — статический сайт, поэтому для публикации не нужен сервер или сборка.

## 1) Загрузить код в репозиторий GitHub

1. Создайте репозиторий на GitHub (например, `price-editor`).
2. Отправьте текущий код в ветку `main`:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/<ВАШ_ЛОГИН>/<ИМЯ_РЕПО>.git
git push -u origin main
```

> Если репозиторий уже есть — просто `git push`.

## 2) Включить GitHub Pages

1. Откройте репозиторий на GitHub.
2. Перейдите в **Settings → Pages**.
3. В блоке **Build and deployment** выберите:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
4. Нажмите **Save**.

Через 1–3 минуты появится ссылка вида:

`https://<ВАШ_ЛОГИН>.github.io/<ИМЯ_РЕПО>/`

## 3) Как открыть редактор прайса

После публикации откройте:

`https://<ВАШ_ЛОГИН>.github.io/<ИМЯ_РЕПО>/price-editor.html`

Или зайдите на главную страницу и нажмите карточку **«Конструктор прайса для сторис»**.

## 4) Обновление сайта

После любого изменения файлов:

```bash
git add .
git commit -m "Update"
git push
```

GitHub Pages автоматически обновит сайт (обычно 1–3 минуты).

## 5) Что удобно делать с телефона

На странице `price-editor.html` можно:
- загрузить фото фона;
- изменить заголовок, контакт и список позиций;
- поменять цвета текста/цен/плашки;
- скачать готовый PNG для сторис (1080×1920).
